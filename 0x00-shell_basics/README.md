0-current_working_directory prints the absolute path name of the current working directory
1-listit displays the content list of the current directory
2-bring_me_home changes the working directory to users home directory
3-listfiles displays the current working directory contents in a long format
4-listmorefiles displays the current directory contents including hidden files in long format
5-listfilesdigitonly displays the current directory contents with user and group id's displayed numerically and hidden files in long format
6-firstdirectory creates a directory called /my_first_directory in the /tmp/ directory
7-movethatfile will move the specified file form the /tmp/ directory to the /tmp/my_first_directory
8-firstdelete deletes the specified file
9-firstdirdeletion deletes the /my_first_directory in the /tmp/ directory
10-back changes the current working directory to the previous directory
11-lists lists all files (including hidden files) within the current working directory, the parent of the working directory and the /boot directory in that order in long format
12-file_type prints the specified file located in the /tmp/ directory
13-symbolic_link creates a symbolic link to the /bin/ls directory
14-copy_html copies all HTML files in the current working directory to the parent directory, unless the file already exists in the parent directory and is not a newer version
